# Math Concepts for Developers: Gradient Descent Explainer

## Overview
This repository contains my final project for the Math Concepts for Developers course. It is a "Math Explainer" focused on the Gradient Descent optimization algorithm. 

The project shows both mathematical theory and practical software implementation, demonstrating how calculus is used to minimize cost functions in programming and machine learning.

## Deliverables
* `gradient_descent_explainer.ipynb`: The primary Jupyter Notebook containing the full technical report, mathematical formulas, Python implementation, and visual experiments.

## Key Features
* **Mathematical Foundation:** Explanations of cost functions, gradients, and the update rule equation.
* **Algorithm Implementation:** A Gradient Descent engine in Python.
* **Visualizations:** Matplotlib graphs mapping the algorithm's step-by-step path to the minimum.
* **Constraint Testing:** Experiments demonstrating the limitations of the algorithm, specifically focusing on divergence (learning rate too high) and slow convergence (learning rate too low).

## How to Run
1. Clone this repository.
2. Ensure you have Python installed along with the `numpy` and `matplotlib` libraries.
3. Open `gradient_descent_explainer.ipynb` using Jupyter Notebook, JupyterLab, or VS Code.
4. Run the cells sequentially from top to bottom.
